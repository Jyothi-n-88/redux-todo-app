# 📝 Redux Todo App

A simple and modern Todo application built using React, Redux Toolkit, and Tailwind CSS.  
This project demonstrates the core concepts of state management using Redux in a practical way.

---

## 🚀 Features

- ➕ Add new tasks  
- ❌ Delete tasks  
- ✅ Mark tasks as completed  
- 🎨 Clean and responsive UI  
- ⚡ Fast performance with Vite  

---

## 🛠️ Tech Stack

- React  
- Redux Toolkit  
- Tailwind CSS  
- Vite  

---

## 📚 Concepts Covered

- Global state management using Redux  
- Actions and reducers  
- Store configuration  
- useDispatch & useSelector hooks  
- Component-based UI design  

---

## 📌 Purpose

This project was built as part of my learning journey to understand how Redux works and how it helps in managing application state efficiently in React applications.
